# BashBox

BashBox is a Linux job-simulation lab platform project.

This repository consolidates the old BashBox codebases into one monorepo for analysis and refactoring.

## Structure

- `apps/frontend` - old frontend project
- `apps/backend` - old backend project
- `archive/bashbox-temp` - old temporary/frontend prototype
- `docs` - planning and documentation

## Current Goal

Analyze the old codebase and gradually refactor it toward the planned BashBox MVP.
